# Lab 02: Queues

This lab covers the implementation of various Queue data structures.

## Contents

- **[CircularQueue](./CircularQueue)**: Implementation of a Circular Queue using an array.
- **[DoubleEndedQueue](./DoubleEndedQueue)**: Implementation of a Double Ended Queue (Deque).
- **[QueueLinkedList](./QueueLinkedList)**: Implementation of a Queue using a Linked List.

## Usage

Navigate to each directory to verify the implementation and run the main programs to see the data structures in action.
